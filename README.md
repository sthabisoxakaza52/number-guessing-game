## Number Guessing Game 

## Description
A simple Python game where the computer picks a random number between 1 and 1000, and the player has to guess it. After each guess, the game gives feedback: 
"Too high"
"Too low"
"Correct!"

This project demonstrates Python basics such as functions, loops, conditionals, random numbers.

## How to play
1. Run the game using Python:
python number_guess.py

2. The computer will pick a random number between 1 and 1000.

3. Enter your guesses when prompted.

4. Keep guessing until you find the correct number.

5. Feedback will be displayed after each guess.

## Features
1. Random number generation between 1 and 1000.

2. Feedback for each guess: too high, too low  or correct
